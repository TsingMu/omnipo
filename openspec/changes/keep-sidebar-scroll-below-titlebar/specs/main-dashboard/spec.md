# main-dashboard 增量规范

## MODIFIED Requirements

### Requirement：侧栏滚动视口必须永久避开标题栏

系统必须将原生 macOS 侧栏的可滚动视口及分组标题限制在窗口标题栏安全区域下方。标题栏避让区域不得作为可随列表滚动消失的内容 margin；当 SwiftUI 安全区不足以表达统一工具栏高度时，系统必须依据窗口实际内容布局自适应，且不得写死标题栏高度。

#### Scenario：切换到非首个侧栏入口

- **当** 用户选择快速启动、剪切板或其他非首个入口
- **那么** 系统可以自动滚动列表以显示选中行
- **并且** 顶部“总览”行不得进入标题栏或窗口控制按钮区域
- **并且** 标题栏安全区不会随滚动内容移出视口

#### Scenario：连续切换入口

- **当** 用户在多个侧栏入口之间连续切换
- **那么** 所有侧栏行的文字和点击区域始终位于标题栏下方
- **并且** 原生选择、键盘导航和滚动行为保持可用
