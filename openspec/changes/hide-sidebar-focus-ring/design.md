# Design：隐藏侧栏外层焦点描边

## 方案

继续让侧栏 `ScrollView` 使用 `focusable()` 接收方向键命令，并在同一容器上关闭 SwiftUI 的默认焦点效果。选中入口仍通过现有蓝色圆角背景表达，辅助功能选中状态保持不变。

## 取舍

- 使用 SwiftUI 原生焦点效果控制，避免扩大 AppKit 边界。
- 仅关闭容器外框，不关闭焦点能力，确保 `onMoveCommand` 继续工作。
- 不为鼠标点击增加程序化聚焦，避免恢复此前的滚动或布局副作用。

## 验证

- 编译并运行真实 macOS 应用。
- 检查侧栏获得键盘焦点后没有蓝色外框。
- 验证上下方向键仍按顺序切换入口。
- 执行完整测试。
