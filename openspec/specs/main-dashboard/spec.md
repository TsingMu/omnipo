# main-dashboard Specification

## Purpose

定义 Omnipo 主窗口的稳定导航与只读总览体验：以分组侧栏组织八个固定功能入口，在 Dashboard 展示产品定位、启动卷容量摘要和安全说明，并确保标题栏边界、键盘导航、窗口缩放以及浅色和深色外观下的可读性。Dashboard 与快捷入口只负责状态展示和导航，不自动启动扫描、清理、删除或其他敏感操作。

## Requirements
### Requirement: 主窗口必须提供清晰的分组导航

系统 MUST 为八个稳定目的地提供位置稳定的分组侧栏，按概览、效率工具和系统工具展示中文标题、语义图标与简短说明，并为 Dashboard 和全部功能详情页提供统一的标题栏安全边界。选择变化不得导致原本可见的侧栏内容整体上下位移，任何详情内容不得进入窗口工具栏区域；入口选择只导航，不得启动敏感业务操作。

#### Scenario: 连续点击全部入口

- **当** 用户依次点击总览、快速启动、剪切板、磁盘清理、应用卸载、权限审计、微信管理和系统监控
- **那么** 未发生手动滚动时侧栏分组与入口保持固定坐标
- **并且** 只有选中高亮和详情内容发生变化
- **并且** 当前入口仍具有明确的选中视觉和辅助功能状态

#### Scenario: 显示任一详情页

- **当** 主窗口显示任一稳定目的地
- **那么** 页面标题、图标、说明、卡片和操作均完整位于工具栏下方
- **并且** 页面内容不会被标题栏裁切

#### Scenario: 使用键盘或小窗口导航

- **当** 用户使用方向键切换入口，或窗口高度不足以显示全部入口
- **那么** 选择按稳定目的地顺序移动并在首尾停止
- **并且** 侧栏可以滚动访问全部入口
- **并且** 仅键盘导航到不可见目标时允许按需滚动

### Requirement: Dashboard 必须说明产品定位与当前状态

Dashboard MUST 展示 Omnipo 品牌说明、启动磁盘状态卡和安全说明。系统在应用启动后必须尽快以只读方式读取启动卷容量摘要，并在可用时显示已用空间、可用空间和总容量；当容量摘要不可用时，系统必须明确显示原因，不得展示虚构容量、占用率或节省空间。

#### Scenario: 首次查看 Dashboard 且容量摘要可用

- **当** 用户打开 Dashboard
- **并且** 系统已成功读取启动卷容量摘要
- **那么** 状态卡显示已用空间、可用空间和总容量
- **并且** 这些数字来自卷级只读元数据
- **并且** 不读取用户文件内容或启动目录扫描

#### Scenario: 首次查看 Dashboard 时仍在读取容量摘要

- **当** 用户打开 Dashboard
- **并且** 容量摘要尚未返回
- **那么** 状态卡显示加载中状态
- **并且** 三项容量字段使用非数字占位符
- **并且** 主窗口其余内容保持可用

#### Scenario: 容量摘要不可用

- **当** 用户打开 Dashboard
- **并且** 系统暂时无法读取启动卷容量摘要
- **那么** 状态卡显示容量信息不可用及原因
- **并且** 不展示演示容量、占用百分比或虚构节省空间
- **并且** 不回退为递归扫描整个磁盘

### Requirement: Dashboard 快捷入口必须保持纯导航

Dashboard 中的快捷入口 MUST 只负责将用户导航到对应目的地。容量摘要状态变化、刷新或降级不得让快捷入口自动触发扫描、清理、删除或其他副作用行为。

#### Scenario: 点击首页快捷入口

- **当** 用户在 Dashboard 点击“磁盘清理”或其他快捷入口
- **那么** 系统仅执行导航
- **并且** 不启动任何目录扫描、清理、删除或后台任务

### Requirement: Dashboard 必须适配 macOS 外观与窗口尺寸

Dashboard MUST 使用系统语义色和材质，并在应用最小窗口尺寸下保持内容可读和可访问。

#### Scenario: 缩小窗口或切换外观

- **当** 用户将窗口缩小到支持的最小尺寸或切换浅色与深色外观
- **那么** 内容可通过自适应布局或滚动继续访问
- **并且** 文字、卡片与操作保持足够对比度

### Requirement: 侧栏必须提供清晰且不重复的焦点反馈

系统 MUST 保留侧栏键盘导航能力，并以入口选中高亮作为主要状态反馈。侧栏滚动容器获得键盘焦点时不得额外显示包围整个侧栏的系统焦点描边。

#### Scenario: 侧栏获得键盘焦点

- **当** 用户通过键盘与侧栏交互
- **那么** 整个侧栏外围不显示蓝色焦点框
- **并且** 当前入口仍显示明确的选中高亮
- **并且** 上下方向键仍按稳定顺序切换入口
